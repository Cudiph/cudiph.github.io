Static module functions

🔗 cuteful.kbd.click (keycode, kbd)
Perform a key press and release.

Parameters:

Name Type(s) Description
keycode integer Event code from linux input-event-codes.h
kbd Optional cwc_kbd The keyboard object

See also:

type Perform sequence of click over given chraacters. static module functions

Usage:

    local cful = require("cuteful")
cful.kbd.click(cful.kbd.event_code.KEY_ENTER)
🔗 cuteful.kbd.type (words, kbd)
Perform sequence of click over given chraacters.

Only support US layout with limited char available.

Parameters:

Name Type(s) Description
words string Characters to type
kbd Optional cwc_kbd The keyboard object

See also:

cwc.kbd.update_modifiers Update modifiers state of this keyboard change are absolute and not incremental. (cwc.kbd) object methods

Usage:

    -- example to type "Hello, world!"
local mykbd = cwc.kbd.get()[1]

mykbd:update_modifiers(0, 0, mod.SHIFT)
cful.kbd.type("h")
mykbd:update_modifiers(0, 0, 0)
cful.kbd.type("ello, world")
mykbd:send_key(cful.kbd.event_code.KEY_LEFTSHIFT, enum.key_state.PRESSED)
cful.kbd.type("1")
mykbd:send_key(cful.kbd.event_code.KEY_LEFTSHIFT, enum.key_state.RELEASED)
🔗 cuteful.kbd.set_layout_by_shortname (words, kbd)
Set xkb_layout option by the shortname specified in the config.

Parameters:

Name Type(s) Description
words string Characters to type
kbd Optional cwc_kbd The keyboard object

See also:

cwc.kbd.layout_index Current active xkb layout index (0-based). (cwc.kbd) object properties

Usage:

    config.init({ xkb_layout = "us,de,fr"})

local cful = require("cuteful")
cful.kbd.set_layout_by_shortname("fr")
