Module: cwc_tag

Tag object for managing virtual workspace in a screen.

To get the tag from the screen use cwc.screen:get_tag.

The screen has its own tags ranged from 1-30, to query the maximum workspace the compositor can handle use cwc.screen.get_max_workspace. Tag index 0 is used for wallpaper view and will not show any client

Object properties

index integer Index of the tag in the screen. Read only
screen cwc_screen The screen where this tag belongs. Read only
selected boolean True if the tag is selected to be viewed.
gap integer The gap (spacing, also called useless_gap) between clients.
mwfact number Master width factor.
layout_mode integer The layout mode of the selected workspace.
master_count integer Set the number of master windows.
column_count integer Set the number of columns.

Object methods

:toggle () Toggle selected tag.
:view_only () Set this tag as the only selected tag.
:strategy_idx (idx) Change the strategy in layout mode.
